​Project Overview
​This repository contains the practical component of the Capstone Project for the OSS course. It includes five functional shell scripts that demonstrate Linux system administration and automation skills, specifically focused on the Python ecosystem.  
​Script Descriptions
​Each script must be executed on a Linux environment.  
​script1.sh (System Identity Report): Introduces the Linux system, displaying the kernel version, current user, and system uptime. It also notes the open-source license covering Python.  
​script2.sh (FOSS Package Inspector): Checks if the python3 package is installed on the system and provides a short philosophical description of the project.  
​script3.sh (Disk and Permission Auditor): Loops through essential system directories (and Python library paths) to report disk usage and file permissions.  
​script4.sh (Log File Analyzer): Scans a specified log file to count occurrences of keywords like "ERROR" or "python."  
​script5.sh (Open Source Manifesto Generator): An interactive script that generates a personalized philosophy statement based on user input and saves it to a text file.
